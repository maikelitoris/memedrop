Track how many consecutive days the user has opened at least one drop.

Data model additions to SharedPreferences:
```json
{
  "current_streak": 3,
  "longest_streak": 7,
  "last_streak_date": "2024-04-19"  // UTC date string "YYYY-MM-DD"
}
```

Streak logic (implement in drop_service.dart):
- On every successful drop open: check if last_streak_date == yesterday (UTC).

**Feature 7 — Streak Freeze (Passive, No UI)**

If the user misses exactly 1 day (gap == 2 days between last_streak_date and today), 
grant one automatic streak freeze per 7-day period:

Logic in drop_service.dart:
```dart
// On drop open, before streak calculation:
final dayGap = today.difference(lastStreakDate).inDays;
if (dayGap == 2) {
  final freezesUsedThisWeek = prefs.getInt('freezes_used_this_week') ?? 0;
  final weekStart = // Monday of current UTC week
  final lastFreezeWeek = prefs.getString('last_freeze_week') ?? '';
  
  if (lastFreezeWeek != weekStart.toIso8601String().substring(0,10) 
      || freezesUsedThisWeek < 1) {
    // Grant freeze: treat as if yesterday was played
    // Don't reset streak, increment normally
    prefs.setInt('freezes_used_this_week', freezesUsedThisWeek + 1);
    prefs.setString('last_freeze_week', weekStart...);
  } else {
    // No freeze available, reset streak
    currentStreak = 1;
  }
}
```

Show a one-time toast/snackbar when a freeze is consumed: 
"STREAK SAVED. 1 FREEZE USED THIS WEEK." — color: #00D4AA (rare cyan), 
displayed for 3 seconds, positioned above the bottom safe area.
